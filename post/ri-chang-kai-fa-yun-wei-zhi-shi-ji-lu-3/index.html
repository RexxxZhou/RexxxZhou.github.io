Linux 命令行快捷键

涉及在linux命令行下进行快速移动光标、命令编辑、编辑后执行历史命令、Bang(!)命令、控制命令等。让basher更有效率。

  • 常用

  1. ctrl+左右键 : 在单词之间跳转

  2. ctrl+a : 跳到本行的行首

  3. ctrl+e : 跳到页尾

  4. Ctrl+u ：删除当前光标前面的文字 （还有剪切功能）

  5. ctrl+k ：删除当前光标后面的文字(还有剪切功能)

  6. Ctrl+L：进行清屏操作

  7. Ctrl+y : 粘贴Ctrl+u或ctrl+k剪切的内容

  8. Ctrl+w : 删除光标前面的单词的字符（以空格隔开的字符串）

  9. Alt – d ：由光标位置开始，往右删除单词，往行尾删

  10. Ctrl+r : 搜索执行过的命令

  11. ! + 最近执行过的命令中的某个字符 ：可快速执行最近执行过的命令

  12. fs -l :显示部分历史命令

说明

  • Ctrl – k: 先按住 Ctrl 键，然后再按 k 键；

  • Alt – k: 先按住 Alt 键，然后再按 k 键；

  • M – k：先单击 Esc 键，然后再按 k 键。

编辑命令

  • Ctrl – h ：删除光标左方位置的字符

  • Ctrl – d ：删除光标右方位置的字符（注意：当前命令行没有任何字符时，会注销系统或结束终端）

  • Alt – d ：由光标位置开始，往右删除单词。往行尾删

WSGI 服务器网关接口

wsgi web server gateway interface

是为python语言定义的服务器与应用程序之间的简单而通用的接口，WSGI区分为两个部分：一为“服务器”或“网关”，另一为“应用程序”或“应用框架”。在处理一个WSGI请求时，服务器会为应用程序提供环境信息及一个回调函数（Callback Function）。当应用程序完成处理请求后，透过前述的回调函数，将结果回传给服务器。

https://zhuanlan.zhihu.com/p/95942024

简单来说，django和tornado是实现了wsgi的框架，整体逻辑就是接受一个请求对象（例如HttpRequest），然后执行函数，通过回调函数，将结果（例如HttpResponse）回给用户。

WSGI接口定义非常简单，它只要求Web开发者实现一个函数，就可以响应HTTP请求。

像tornado中定义的Class中的get、post成员函数，django中views.py中定义的函数。是一种框架层面的设计，开发者只需要关注业务函数。

elasticsearch 的scroll

es查询回包中，有hits.totals字段，这个值表明了我们需要匹配的条件总共查到了有多少个文档记录，这个值不是我们指定的size大小，size是确切要es这次返回的结果数量，是hits.hits数组的最大值

scroll是像卷轴一样，每次返回多少个结果，假如scroll的大小是10000，那么每次最多返回10000个数据，像卷轴一样慢慢滚动返回，使用之前的查询参数再次请求es。类似于让es做分页了一样。python的ElasticSearch库就有scroll这个函数，就是实现这个功能的。

无服务器架构

https://www.redhat.com/zh/topics/cloud-native-apps/what-is-serverless#%E6%A6%82%E8%BF%B0

https://jimmysong.io/kubernetes-handbook/usecases/serverless.html

Serverless（无服务器架构）指的是由开发者实现的服务端逻辑运行在无状态的计算容器中，它由事件触发， 完全被第三方管理，其业务层面的状态则被开发者使用的数据库和存储资源所记录。

简单来说，就是开发人员不需要关心底层设备的扩缩容和管理，只需要关心自己写的业务逻辑，由架构本身来保证你的代码可以执行和响应请求，通过容器编排，来执行你的业务逻辑代码。

是在PaaS上更抽象的一层；只要是开发人员使用时，不需要去关心底层设备和资源的，都可以看作是无服务架构。

mysql noexists用法

insert into tableA select * from tableB where 
noexists(select * from tableB where tableA.id=tableB.id);

这个语句其实就是将tableB的数据导入到tableA中，但是只导入tableA中没有的记录；

逻辑是，以tableB（被导入表）的记录去匹配tableA的记录，如果有，说明tableA中已经有了这条记录，那么对应的tableB的这条记录不会被插入到tableA中，反之则会插入。

excel的vlookup函数

这个函数是用来匹配查找的，根据某行某个值来查找然后获取对应命中的行的一些字段值

https://zhuanlan.zhihu.com/p/29161495

=VLOOKUP(D823,sheet_name!$A1:1:1:G$7063,2,FALSE())

D823是要被匹配内容，sheet_name是要被匹配的表，$A1:1:1:G$7063这是一个区域，表示从A列的第一行到G列的7063行这个矩形区域内，去查找D823格的内容，FALSE()表示精确匹配，否则为模糊匹配，数字2表示刚刚说的区域的第几列，这个意思是说，如果在这个区域匹配到的某一行，选择哪一列的值作为结果值。

这里特意记录下局部静态变量的事，全局静态变量和全局变量都比较熟悉。

局部静态变量和全局静态变量没有什么区别，只是全局的话，在函数进入main的时候已经在数据段或者BSS段（为初始化全局数据段）中生成了对象，而函数内部的局部静态变量，则要到执行函数的时候才会生成。

但是静态变量的特性不变，唯一和共享，直到函数结束才会释放内存。
